# ClearTab - Custom Homepage

A minimalist, distraction-free custom homepage Chrome extension inspired by Glassmorphism design. Replace your default new tab page with a personalized dashboard featuring search, quick links, clock, calendar, and more.

## Features

- **Glassmorphism Design**: Modern, translucent UI with blur effects and elegant styling.
- **Multi-Engine Search**: Switch between Google, YouTube, and DuckDuckGo search engines.
- **Personalized Greeting**: Dynamic greeting based on time of day, with editable user name.
- **Quick Access Cards**: Add and manage custom links to your favorite websites with favicons.
- **Live Clock & Date**: Real-time clock display with current date.
- **Calendar Widget**: Navigate through months and view dates with past/present/future styling.
- **Dark/Light Mode**: Toggle between themes with smooth transitions.
- **Dynamic Backgrounds**: Automatically fetches high-quality images from Picsum (online) or uses offline fallback.
- **Privacy-Focused**: No tracking, uses local storage for settings.

## Installation

1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" in the top right.
4. Click "Load unpacked" and select the `CustomHomePage` folder.
5. The extension will replace your new tab page with ClearTab.

## Usage

- **Search**: Type in the search bar and select your preferred engine using the toggle icon.
- **Add Cards**: Click the "+" button to add new quick access links.
- **Edit Name**: Click on your name in the greeting to edit it.
- **Theme Toggle**: Use the Dark-Mode/Light-Mode button to switch themes.
- **Calendar**: Navigate months using the Prev/Next buttons.
- **Settings**: Access additional settings via the gear icon (if implemented).

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (ES6+)
- Chrome Extension Manifest V3
